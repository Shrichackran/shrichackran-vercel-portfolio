# Shrichackran K M — Interactive 3D Developer Portfolio

A production-ready interactive developer portfolio built with HTML, CSS, vanilla JavaScript, and Three.js.

## Highlights

- Interactive 3D background using Three.js
- Professional animated loading screen
- Recruiter-friendly sections: About, Tech Stack, Projects, Experience, Education, Contact
- Real project galleries for Store Intelligence, LifeLink, Speech Denoising, and JoBzz
- WiFi Network Controller included without project assets as requested
- EmailJS contact form using the provided service/template/public key
- Vercel-ready static build setup

## Local preview

```bash
python3 -m http.server 8000
```

Open the forwarded/local URL.

## Vercel build

```bash
npm run build
```

Output directory: `dist`
